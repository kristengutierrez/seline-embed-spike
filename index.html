Live Embed Spike

Test harness for: can a self-hosted page render as a LIVE embed inside a published Canva website?

  • /plain — control, no embed metadata
  • /oembed — H1: oEmbed discovery link
  • /player — H2: Twitter player card
  • /og — H3: Open Graph video/iframe
  • /widget — the live payload (ticking clock + countdown)
